📦 Lists

Created lists for letters, numbers, and symbols

Appended selected characters to a new list using append()

🔢 For Loops & Range

Used for loops to repeat tasks a specific number of times

Used range() to loop through the desired number of letters, numbers, and symbols

💻 Project: Password Generator

🎯 What This Program Does

Asks the user how many letters, numbers, and symbols they want in their password

Randomly selects characters from predefined lists

Shuffles the selected characters to ensure randomness

Converts the list of characters into a single string and prints the password
